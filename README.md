# Week 1 프로젝트 개요

# 문제 배경 및 프로젝트 개요

혼자 사는 사람이나 1~2인 가구는 냉장고 안에 식재료가 남아 있어도, 그 재료들로 어떤 음식을 만들 수 있는지 바로 떠올리지 못하는 경우가 많아 다음과 같은 문제가 반복된다.

1. 냉장고 안의 식재료를 다 쓰지 못하고 버리게 된다.
2. 이미 있는 재료를 또 사게 된다.
3. 레시피를 찾아도, 내가 가진 재료로 가능한지 바로 판단하기 어렵다.
4. 요리하기 전 “뭘 먹지?”를 정하는 데 시간이 오래 걸린다.

"오늘냉장고"는 이 문제를 해결하기 위해 기획되었다.
사용자는 재료를 관리하고, 시스템은 그 재료를 바탕으로 추천을 제공한다.
요약하자면, 이 서비스는

1. "식재료 관리 서비스"이면서 동시에
2. "레시피 추천 서비스"이다.

## 프로젝트 목표

["서비스" 로서의 목표]

1. 사용자의 냉장고 재고를 중심으로 식사 결정을 돕는다.
2. 유통기한 임박 재료를 먼저 소비하게 한다.
3. 중복 구매와 음식물 낭비를 줄인다

["프로젝트"로서의 목표]

1. 공공 레시피 API를 활용하여 현재 만들 수 있는 요리를 추천한다.
2. 부족한 재료를 자동으로 추출, 구매링크를 만들어 향후 수익화를 도모한다.
3. 레시피 상세 화면에서 조리 순서와 영양 정보를 직관적으로 확인 가능하도록
한다.
4. 북마크와 후기 기능을 통해 서비스를 반복적으로 사용할 수 있는 구조로 만든다.
5. 모든 팀원이 각자 맡은 도메인을 풀스택으로 구현하도록 설계한다.

## 주요 사용자

1. 자취생
2. 1~2인 가구
3. 장은 보지만 재료 관리가 체계적이지 않은 사용자
4. 냉장고 재고를 활용해 식비를 아끼고 싶은 사용자
5. 유통기한이 임박한 식재료를 효율적으로 소비하고 싶은 사용자

**`즉, 요리를 아주 전문적으로 하는 사람보다는,
"실생활에서 빠르게 식사를 결정하고 싶은 일반 사용자"가 주요 타겟이다`**

## 서비스 핵심 기능

| 홈 페이지
  • 오늘냉장고의 출발점.
  • 오늘의 추천 요리, 검색창, 공통 내비게이션 바, 로그인/회원가입 기능.
  • 서비스 전체 분위기와 진입 경험을 결정하는 페이지 |
| --- |
| 현재 냉장고 페이지
  • 사용자 보유 식재료 등록, 삭제, 조회 가능.
  • 각 식재료는 이름, 유통기한이 함께 저장. 유통기한 가까운 항목은 시각적으로 강조됨. |
| 추천 레시피 페이지 (핵심 기능)
  • 사용자 재료 & API의 레시피 재료 비교, 재료 일치율 높은 레시피 우선 추천.
  • 재료 부족 시 시각적 표시 & 쇼핑링크로 연결.
  • 동의어 사전 기능. (예 : 계란/달걀, 파/대파) |
| 레시피 상세 페이지
  • 선택한 음식의 조리 단계, 이미지, 칼로리와 영양 정보 제공
  • 타이머 포함하여 사용 편의성 향상 |
| 북마크 & 소셜 페이지
  • 사용자는 마음에 드는 레시피를 저장할 수 있고, 실제로 만든 요리를 후기와 이미지로 남길 수 있다.
  • 이 기능은 단순 검색 서비스에서 끝나지 않고, 사용자가 계속 머무르고 기록을 쌓게 하는 역할을 한다. |

## 시스템 종합 흐름도

```
[비회원]
 ↓
[홈]
   ├─ [레시피 검색] > [전체 검색 레시피 목록 확인]
   ├─ [회원가입]
   └─ [로그인]
          ↓
       [회원 홈]
          ↓
   ┌──────┼───────────────┬───────────────┐
   ↓      ↓               ↓               ↓
[냉장고] [레시피 추천]   [전체 레시피]     [북마크]
   ↓      ↓               ↓               ↓
재료관리   추천 목록 확인   검색/탐색    저장 목록 확인
   ↓      ↓               ↓               ↓
임박재료   상세 페이지 이동 ──────────→ [레시피 상세]
알림 확인                                  ↓
                                     [북마크 추가]
                                          ↓
                                    [소셜 후기 작성]
                                          ↓
	                                  [소셜 게시판 확인]
```
